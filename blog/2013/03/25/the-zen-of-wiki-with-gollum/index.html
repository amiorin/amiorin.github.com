The Zen of Wiki With Gollum

I’ve always loved wikis. They are very useful for bookmarks and snippets. After many years of Dokuwiki, I switched to Evernote. The Evernote experience was very disappointing. But the Dokuwiki solution is based on Lamp that is so old and I didn’t want to go back.

So I switched to Gollum and now I’m very satified.

  • I can use Vim to edit my wiki.
  • I can use Ack to search in my wiki.
  • I can edit it on Github, if I need to.

Conclusion

  • No database.
  • Password protected if you need.
  • Distribuited.
  • Autosync.
  • It works offline.
  • Vim editing instead of the textarea.
  • Regex search with Ack.
  • Instant preview in the browser.
  • Markdown language like in Jekyll.

I’m ok for the next 10 years. :-)
